Introducción

A modo de introducción hay tres articulos: El primero define Machine Learning más a fondo y los separa de data mining, análisis estadístico, etc. El segundo explica la metodología CRSIP para proyectos de análisis de datos y el tercer artículo define herramientas de Machine Learning y compara la disponibilida de algunos operadores

Tratamiento previo de datos

En Machine Learning hay una máxima muy utilizada: "El 80% del trabajo se basa en preparar los datos". Es crucial para que Machine Learning sea útil que primero se entienda el contexto del negocio, luego de los datos y finalmente se preparan los datos para ser utilizados. Este proceso es crucial debido a que un modelo basado en fallos interpretativos no sirve de nada y probablemente brinde información errónea perjudicando a la organización.
